Before your first scan

Questions from the field.

TrailEye AI is a Windows desktop application. Your files remain in their existing folders while the app builds a local, searchable record.

Does TrailEye upload my photos or videos?

Not for the standard local detection pipeline. Animal, person, vehicle and species detection run on your PC. The separate custom text-prompt cloud search is optional, requires your own provider API key and clearly indicates that it sends images to that provider.

Can it analyze video as well as photos?

Yes. TrailEye can analyze both trail-camera images and videos in the same folder workflow.

What can the AI recognize?

The local pipeline detects animals, people and vehicles, then refines animal detections into a library of 38 wildlife species where possible.

What happens if the timestamp is burned into the image?

TrailEye can read timestamp overlays with OCR. It also uses EXIF metadata and file time as fallbacks, with a confidence value shown in the event details.

Is a subscription required?

No. Free is permanent. Explorer and Professional are one-time purchases, activated with a licence key.

Which edition should I choose?

Use Free to test up to 20 files per analysis. Explorer is designed for one camera folder and up to 1,000 files per run. Professional removes file limits and includes subfolder scanning for complete multi-camera archives.
